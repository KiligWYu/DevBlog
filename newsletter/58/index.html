Kilig 的碎周报 - #58（2023.04.04）

2023-04-04 | 碎周报

Hi There! 感谢阅读 Kilig 的碎周报第 58 期。本篇是对过去一周（03.27 - 04.03）的记录、思考与分享。
本期碎周报晚了一天更新。

生活

跟上周一样，在小妮姥姥家。
周末吃了巴奴火锅，打包带回家吃的，好久没吃火锅了。巴奴的打包也太慢了，锅底最难打包了，用塑料袋装，太容易洒了。相比海底捞打包更方便些。
3 号姥姥家给小妮办了百岁宴。上个月底在老家也给办了，不过我们没回去 😅。满月时因为疫情及天冷没有办。

产品动态

更新了花密的 iOS v1.6.0 和 macOS v2.10.0，更新内容见下方。

花密是第三方客户端，官网在这里，官方很早之前是有 iOS 和 macOS 客户端的，但后来很久不更新也下架了，原作者把花密的算法开源了，因为花密也是我要经常要使用的工具，所以干脆我来做个花密客户端。
我是花密的老用户了，在使用 macOS 之前，花密就是我的主力密码管理工具，花密非常轻量，尽管还是要记一些区分代号；工作以后使用 macOS 了，也使用过像 1Password 这样的密码管理工具，但每次输保险库密码，也很麻烦并不方便，就又使用回了花密；再后来使用 iCloud 密码 + 花密辅助。
密码管理工具有很多，适合自己的才是最好的。

花密 for iOS

  • 新增：设置区分代号后缀
  • 新增：AccentColor
  • 优化：支持滑动时隐藏键盘了，需要 iOS 16
  • 优化：其他代码上的小修改和优化

花密 for macOS

  • 新增：设置区分代号后缀
  • 新增：AccentColor
  • 优化：开机启动，使用 macOS 13 提供的新方法
  • 优化：快捷键设置
  • 优化：保存密码到钥匙串
  • 优化：设置页面 UI
  • 优化：其他代码上的小修改和优化
