Kilig 的碎周报 - #64（2023.05.15）

2023-05-15 | 碎周报

Hi There! 感谢阅读 Kilig 的碎周报第 64 期。本篇是对过去一周（05.08 - 05.14）的记录、思考与分享。

  • 周一（05.08）发完周报，又更新了下博客主题，主要修改了下字体，Chrome 加载网页时可以使用电脑上已安装的字体，Safari 就不行。不久前把博客字体改成了 LXGW Wenkai Screen，只是简单改了下 CSS 的 font-family，这次使用了 lxgw-wenkai-screen-webfont 方案，这下在 Safari 以及移动端都能显示落霞孤鹜字体了。
  • 最后两张星巴克优惠券快到期了，用了，以后就尽量不买星巴克的咖啡了（除非忍不住， 除非有很喜欢的星礼包），星巴克的咖啡贵但品质一般。等我囤的三顿半喝完了，就试试 illy 的咖啡粉，自己冲。
  • Raycast 推出了 Pro，而我又用回了 Alfred，并整理了下 Alfred 的 Workflows，把之前的 Workflows 除了自己写的都删了，从官方的 Alfred Gallery 里安装了些。
  • 再给自己加一条：无非必要出门不开车。
  • 这几天刷推，感觉所有人都在玩 塞尔达传说：王国之泪，除了我 🤣。我现在最期待的游戏是 丝之歌(鸽)。

买买买

  • 在 第 36 期的碎周报 里分享过 MediaMate，一款漂亮的替换系统声音、亮度 HUD 的 App，现在终于还是买了。€4.99 合 $5.45。
  • 买了 死亡细胞 的新 DLC 重返恶魔城，DLC 刚出的时候就想买来着，忙忘了，虽然手残加没空玩，但先收了。
  • 买了新域名 deardiary.best，新产品影儿都没有，域名先行 😂️；看域名就能猜出来新产品是关于日记的，对；虽然日记产品也非常卷，但我这款跟其他的又有些不同，更多信息等我弄出来再说吧，希望不鸽（咕咕咕……）。
