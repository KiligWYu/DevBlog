Kilig 的碎周报 - #61（2023.04.24）

2023-04-24 | 碎周报

Hi There! 感谢阅读 Kilig 的碎周报第 61 期。本篇是对过去一周（04.17 - 04.23）的记录、思考与分享。

日常跟之前一样，娃儿占据了大部分时间，剩下的用来阅读、学习和娱乐。

目前在使用的邮件服务主要是 Outlook 和 Gmail，分别使用 Outlook 和 Mimesteam 客户端，之前一直在用系统自带的 Mail 客户端，但经常出现账户离线连不上的问题，遂弃用。

大部分应用也使用 Homebrew 覆盖安装了，借助 Homebrew Bundle 可以轻松备份已安装的软件（brew bundle dump --describe ）。

之前是用 CocoaPods 创建的自用 SDK，现在切到了 Swift Package Manager，整理了下笔记里的代码片段，加到了 SDK 里。
