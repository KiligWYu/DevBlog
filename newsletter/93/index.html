Kilig 的碎周报 - #93（2023.12.04）

2023-12-04 | 碎周报

Hi There! 感谢阅读 Kilig 的碎周报第 93 期。本篇是对过去一周（11.27 - 12.03）的记录、思考与分享。

把博客的副标题改成了『代码之外』，博客上发的都是些非技术上的文章，我已经跟自己和解了，承认自己写不出什么高深的技术文章，所以就叫代码之外好了。
另外，也推荐一下 代码之外 Beyond Code 的播客。

代码之外 Beyond Code 是一档由 GeekPlux 和 Randy 共同主持的程序员闲聊播客节目。

读完了《读库 2205》，这本书在我桌子上躺了一年了都，才看完，接下来把《读库 2206》也看了。22 年订了一年的读库会员，23 年没有再续订，一是为了开源节流，另外我也不缺其他读物。

在 Sketch 里重新设计了简历，之前是用 冷熊简历 等制作的 Markdown 格式的简单简历，重新设计简历后，把原来两页的松散简历压缩到一页，信息量不减反增，并且设计的还很好看。
在设计上参考了 简单简历，简单简历是个很棒的在线制作简历的工具，有很多模板，可以轻松制作出好看的简历。

本周入门了 TCA 架构，接下来的 SwiftUI 项目也尝试使用 TCA 架构，再推荐下 ethanhuang13 的 iOS 開發配飯吃・SwiftUI + TCA 系列。

本周下午基本都在万达星巴克阅读或工作，图书馆还没装修好，从 8 月开始就闭馆了。下周继续在星巴克办公，也继续我的 开封旅游。
