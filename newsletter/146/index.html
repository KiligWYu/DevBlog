2025-02-17 | 碎周报

Hi There! 感谢阅读 Kilig 的碎周报第 146 期。本篇是对过去一周（02.10 - 02.16）的记录、思考与分享。


前段时间有天我忽然意识到一个问题，我似乎变得冷漠了许多，上班下班，路上常遇到需要帮助的人，我都是匆匆赶路，错过了，我才想起来我可以顺手帮一把的；当我意识到我的冷漠后，我觉得我本不是个冷酷无情的人，我也是有助人为乐的精神的，以后再遇到需要帮助的人，能帮就帮一把。
比如这周有天晚上下班在地铁站，扶梯边有个人拎两个大箱子，不好上扶梯，在扶梯边犹犹豫豫，路过好几个人，我也路过，二话不说（戴着耳机呢），顺手接过一个箱子，顺利上下扶梯。
一点微不足道的小忙，也可以让我心情变好一些。

#差生文具多 我又换终端了，试试 Ghostty，之前使用 Alacritty。两者的差别可以参考 在 Mac 上使用 Ghostty 代替 Alacritty。

Ghostty: 👻 Ghostty is a fast, feature-rich, and cross-platform terminal emulator that uses platform-native UI and GPU acceleration.
Alacritty: A cross-platform, OpenGL terminal emulator.

花了一点点时间，又重构了下 Surge 的配置文件，利用 配置分离，重新整理了下 Mac 和 iPhone 的配置，共用的部分放一个配置文件里，差异化的部分也单独放在不同配置文件里。
代理部分就用订阅配置的代理和分组，之前是手动管理代理和分组，线路太多，去掉无用代理，重新分组，这样比较麻烦，需要定期手动更新，现在直接使用订阅配置里的代理和分组，免维护。
对规则部分进行了拆分，分为自定义规则、通用规则、iPhone 专用规则、Mac 专用规则，规则的更新维护也更清晰方便。
