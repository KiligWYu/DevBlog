2023-03-27 | 碎周报

Hi There! 感谢阅读 Kilig 的碎周报第 57 期。本篇是对过去一周（03.20 - 03.26）的记录、思考与分享。

生活

跟上周一样，还在小妮姥姥家。小妮现在白天睡的少了，晚上睡前会闹一小会，晚上睡的挺好，就是睡的晚，上午起的也晚。

之前订阅了很多的 Newsletter，退订了些偏鸡汤类的、不感兴趣的，剩下的大部分都是技术类的。

日常读 Newsletter 或 RSS，看到需要精读或记笔记的文章就加到 Safari Reading List 里，现在 Reading List 已经累积了好多篇了，之后再一一处理。

阅读

读完了《鞑靼人沙漠》，虽然是豆瓣高分，但个人不是很喜欢，耐着性子读完了。

产品动态

本周为花密 iOS 端和 macOS 端增加了新功能，为区分代号增加后缀，该功能也是用户反馈的。该功能已经开发完成了，顺便更新一下其他部分的代码，之前看的《App 架构》，对 MVC 和 MVVM 有了新的见解，正好实践一下，下周再更新版本。

其他有趣的

  • 真的吐了，VR 眼镜最强劝退指南！【柴知道】
  • 湾区日报 复活了
  • yihong0618/twint: An advanced Twitter scraping & OSINT tool written in Python that doesn’t use Twitter’s API, allowing you to scrape a user’s followers, following, Tweets and more while evading most API limitations.
    twint 已经不维护了，可以使用 yihong 维护的版本。

【完】以上就是本期周报的全部内容了。Have an A-1 week.
