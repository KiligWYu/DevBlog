Kilig 的碎周报 - #147（2025.02.24）

2025-02-24 | 碎周报

Hi There! 感谢阅读 Kilig 的碎周报第 147 期。本篇是对过去一周（02.17 - 02.23）的记录、思考与分享。


  1. 最近改用 Startpage 作为默认的搜索引擎，在配置 Alfred 快速发起 Startpage 搜索时，遇到个问题，Startpage 发起的搜索用的 POST 请求，而 Alfred Web Search 只能用 GET 请求，用 Workflow 的话又该如何实现，我咨询了 DeepSeek 如何用 Alfred 快速发起 Startpage 搜索，DeepSeek 给的方案：创建一个 Workflow，Run Script（如下），在浏览器中打开一个临时的自动提交的表单。

    妙啊，是我未曾想到的方案。但是 AI 的答案我们要“小心求证”，比如这里 DeepSeek 给的提交表单的 action 是 /do/search，而我查了下 Startpage 官网，action 是 /sp/search（我也试了下 do/search 确实也行，但还是用官网的 sp/search）。

  2. 整理了下邮件，该删除的删除，该归档的归档。对于收到的一些如验证码、广告邮件、垃圾邮件等不重要的邮件随手删除，重要邮件定期归档，做好邮箱管理，避免邮件堆积。

  3. #差生文具多 冲颜值买了 IQUNIX MG65 键盘，白色太好看了，官方店铺缺货（现在应该有了），在闲鱼上淘到了好价，¥578，在我的闲鱼的浏览历史里，同款键盘都很快就卖出了，可见这可键盘有多火，价位都在 ¥600+，我刚好刷到了这个成色好还低价的，就果断下手了。到手试用了下，挺好用，我也很喜欢，就是这 65 键盘的布局，没有 F1, F2, etc. 键倒是无所谓，反正平时也几乎用不到，但少了 ~ 和 `，这两个键还是很常用的，需要靠组合键 fn + ⇧ + \ 和 fn + \ 来实现了。
    少数派上有一篇关于此键盘的文章 - iQunix 终于有一款完整适配 macOS 的键盘了！。
    我之前在用的 Keychron K6 红轴的键盘，在蓝牙连接模式下，部分按键经常断触，也就弃用吃灰了。想放闲鱼上出了，但断触的问题我也不确定是键盘本身的问题还是与 macOS 系统兼容性问题还是什么别的问题，如果是键盘本身的问题，那放闲鱼就有点坑买家了，另外我也看了下闲鱼上这款键盘的行情，价格低到几十一百，还无人问津，当废品也不好出，就先继续放着吃灰吧。
