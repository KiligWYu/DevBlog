Kilig 的碎周报 - #72（2023.07.10）

2023-07-10 | 碎周报

Hi There! 感谢阅读 Kilig 的碎周报第 72 期。本篇是对过去一周（07.03 - 07.09）的记录、思考与分享。

  • 在家呆久了，有些烦躁，想去骑行但天太热，想去星巴克办公，但又离不开小可爱，奶奶一个人带不住她……
  • 部署了 RSSHub 到我的丐版 Lightsail，部署完 VPS 就跪了，排除网络问题，后来才发现是撞 CPU 容量墙了，才发现还有 CPU 基准利用率、突增容量这么个玩意儿，之前都没留意过。但也不值得为此更换更贵的 VPS，后来把 RSSHub 部署到了 Vercel 了，部署非常简单方便。
  • 开发完了 Tapet（macOS 上获取 UHD 且无水印且能自动更换桌面壁纸的必应壁纸小工具）的官网 tapet.best，利用 React.js + Tailwind CSS，拖延了许久了，域名也是一早就买了的，之前一度想放弃 🤣，主要是想拿来学习 React。一个简陋的页面，也部署到 Vercel，并绑定了域名。在移动端上访问还有些问题，并未适配，以后再说吧。
  • #TIL macOS Preview.app 导出图片时，按住 ⌥ 点击格式，可以选择更多图片格式。
  • 买了 objccn.io 的新书 《SwiftUI 编程思想 (2023)》。
