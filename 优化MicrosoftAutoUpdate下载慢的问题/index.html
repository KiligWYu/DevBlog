优化 Microsoft AutoUpdate 下载慢的问题

2020-03-16

最近 Microsoft AutoUpdate 下载特别慢，还经常失败，于是在 Console 中看了下它的日志，发现其访问了 azurewebsites.net 和 akamaized.net，下载是从 akamaized.net 这里下载的。于是把这两个域名都加入到某软件的自定义规则中，然后下载就快多了。
