Markdown 中的锚点语法

2018-03-27

Markdown 中的锚点语法：[标题](#标题) ，比如

  • 标题中的空格要替换成连字符 -，如

  • 标题如果含有 .，不同的编辑器或网站语法不太一样，有的要把点替换成连字符（如本站），有的则要去掉（如 GitHub）

  • 标题中的其他字符，如括号引号等，跟点的处理方式差不多，替换成连字符或删掉或不变

    另外，有的编辑器支持用 [TOC] 来直接添加目录，如 MWeb。
