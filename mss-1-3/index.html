MSS 1.3 版本发布：更多使用方式

2023-11-09 | App

MSS（图影） 1.3 版本已经上架 Mac App Store，距离上个版本已经过了半年了，本次更新增加了服务和打开方式，后面有介绍，还修复了快速操作的本地化问题，还新做了张 App Store 里的预览图片。

MSS

MSS，是『Mac Style Screenshot』的缩写，中文名『图影』，是一个给图片添加阴影效果的小工具，模拟系统窗口截图时的阴影效果。

使用

除了常规的打开 MSS 拖拽图片进去的使用方法外，MSS 还支持快速操作、服务、右键打开等方式使用。

快速操作

在访达中选中图片后，在右侧的预览下方可以自定义快速操作，把 MSS 加进去，然后就可以快速地给图片添加阴影，app 都不用打开。
在图片上右键 -> 快速操作 -> MSS 也一样。

在 系统设置 -> 隐私与安全性 -> 拓展 -> 访达 -> 选中 MSS，启用 MSS 快速操作。

服务

MSS 也支持作为服务启动，在 系统设置 -> 键盘 -> 键盘快捷键 -> 服务 -> 图片 -> 选中 MSS，启用 MSS 服务。另外，服务是支持自定义快捷键的，但使用时会打开主程序。

在图片上右键 -> 服务 -> MSS，使用 MSS 服务。

使用 MSS 打开图片

在图片上右键 -> 打开方式 -> MSS。

最后

MSS 还有许多不足之处，阴影也没有系统窗口阴影细腻，但是我自己很喜欢的一个小工具，后续会不断更新维护的，也会一直免费提供。如果你恰好也喜欢 MSS，还请 在 Mac App Store 给个好评，非常感谢！
